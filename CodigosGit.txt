git clone -> traz o repositorio da nuvem para o diretorio local;

status -> ciclo de estado dos meus arquivos;

git add -> adiciona um arquivo pro commit;

git commit -am -> faz um novo bloco de funcionalidade;

git push -> envia o commit para a nuvem;

git checkout -b "Suzy" -> Cria uma nova brench com o nome Suzy "";


Tipos de arquivo:
"untracked" o arquivo nao esta no commit ainda;
"tracked" o arquivo esta no commit;
"modify" o arquivo foi modificado;
"unmodify" o arquivo nao foi modificado.
